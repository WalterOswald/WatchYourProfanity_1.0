Whatch your Profanity

This piece of software was designed as a simple image layout tool for websites.

The inspiration behind this project was a frustrating experience trying to create free-form image layouts using the Cargo site builder. The title is derived from the fact that I personally swear a lot when working with software — especially when it doesn’t work as I expect. Ironically, this software may make you swear a lot if you seriously try to use it. There are many bugs and a lot of jank. Full disclosure: I am not a software engineer.

My personal belief is that any piece of software should aim to restrict its functionality to a small number of core tasks. Overly complex toolboxes often serve to create predefined workflows, which might be beneficial for efficiency, but ultimately kill creative problem-solving.
WYP Sports a limited yet potent collection of features:
1. Freely position your Images with drag and drop.
2. Controll Size and Z-index of your images.
